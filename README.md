# 🧵 VastraVeda

A clothing knowledge platform built with React + Vite + Tailwind CSS.

## Features
15 features including Submission Portal, Tagging, Fabric Substitution, Multilingual support, Media Upload, Badges, Comparison Tool, Debate Board, Issue Tracker, Leaderboard, Dataset Export, Version Control, AI Suggestions, Fabric Identifier, and Open API.

## Tech Stack
- React 18
- Vite
- Tailwind CSS
- React Router DOM

## Setup
```bash
npm install
npm run dev
```
